Adding a Graph

Lets add a graph UI node and take the output from the getBookTicker Node and prepared it so that the graph node can display both the ask price and the bid price on the same graph. We add two Change Nodes and name them isolateBidPrice and isolateAskPrice. Change nodes allow you to take input messages and change the output based on a set of rules. To be able to use the chart properly, we need to deliver a msg.payload with a chartable value, and a msg.topic set to the name of the chartable value.
